Tutorial

"Le damos la bienvenida a la página de tutoriales. Todas nuestras páginas funcionan a través de comandos de voz, y estamos aquí para ayudarle en cada paso. A continuación, le proporcionamos los comandos clave: Para regresar a la página de inicio, por favor diga 'volver a inicio'. Para iniciar la navegación, simplemente utilice 'iniciar'. Si desea pausar la navegación, puede decir 'pausar la navegación'. Para finalizar la navegación, basta con mencionar 'terminar la navegación'. Después de pausar, si desea continuar la navegación, utilice 'continuar navegación'. Si necesita acceder a información adicional, puede decir 'información'. Para retroceder a la página de tutoriales, por favor utilice 'volver a navegación'."
